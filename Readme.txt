
# 🪑 Online Furniture Shop Management System

An all-in-one web-based application designed to manage online furniture sales, inventory, and customer orders efficiently. This project helps both administrators and users to interact with the system to manage and buy furniture products seamlessly.

## 🔧 Features

- 🛒 Browse and purchase furniture products
- 🔍 Product search and category filtering
- 👤 User authentication (Login/Registration)
- 📦 Order management and tracking
- 📊 Admin panel for managing products, categories, orders, and users
- 💳 Payment integration (if implemented)
- 📁 Database integration for data persistence

## 🛠️ Technologies Used

- **Frontend:** HTML, CSS, JavaScript, Bootstrap
- **Backend:** PHP
- **Database:** MySQL
- **Server:** Apache (XAMPP recommended)
